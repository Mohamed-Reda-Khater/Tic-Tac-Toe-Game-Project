# Tic-Tac-Toe – Python GUI Game (Tkinter)

This project is a simple yet polished **Tic-Tac-Toe** game built using **Python** and **Tkinter**.  
You play as **X** against the computer, whose moves are generated randomly.

## Features
- Clean graphical user interface using Tkinter
- Interactive 3×3 game board
- Smart UI updates (colors for win, lose, or draw)
- Live score tracking for both player and computer
- Restart button to reset each round
- Structured and readable code suitable for beginners

## How It Works
1. The player selects a cell to place **X**.
2. The computer responds with a random **O** move.
3. The game detects wins, losses, and draws.
4. Winning cells are highlighted:
   - **Player X win → Cyan**
   - **Computer O win → Red**
5. Draw → Entire board turns red.
